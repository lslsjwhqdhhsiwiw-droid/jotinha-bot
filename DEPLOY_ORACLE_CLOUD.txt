⚙️ PASSO 7: CONFIGURAR O BOT
═══════════════════════════════════════════════════════════════

1. Na VM, navegue até a pasta do bot:
   cd discord-bot-jotinha  (ou nome da sua pasta)

2. Crie ambiente virtual Python:
   python3 -m venv venv
   source venv/bin/activate

3. Instale as dependências:
   pip install -r requirements.txt

   OU se não tiver requirements.txt:
   pip install discord.py[voice] yt-dlp python-dotenv flask beautifulsoup4 requests trafilatura wavelink aiohttp psutil pytz pynacl

4. Configure as variáveis de ambiente:
   nano .env
   
   Cole seu token:
   DISCORD_TOKEN=SEU_TOKEN_AQUI
   OPENAI_API_KEY=SUA_CHAVE_AQUI
   
   Salve: Ctrl+X, depois Y, depois Enter

═══════════════════════════════════════════════════════════════
🚀 PASSO 8: RODAR O BOT 24/7
═══════════════════════════════════════════════════════════════

MÉTODO 1 - Usando tmux (Simples):
----------------------------------
1. Instale tmux:
   sudo apt install tmux -y

2. Crie uma sessão:
   tmux new -s bot

3. Rode o bot:
   python3 bot.py

4. Desconecte sem parar o bot:
   Aperte: Ctrl+B, depois aperte D

5. Para reconectar depois:
   tmux attach -t bot

MÉTODO 2 - Usando PM2 (Mais profissional):
-------------------------------------------
1. Instale Node.js e PM2:
   curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
   sudo apt install -y nodejs
   sudo npm install -g pm2

2. Crie arquivo de inicialização:
   nano start.sh
   
   Cole:
   #!/bin/bash
   cd /home/ubuntu/discord-bot-jotinha
   source venv/bin/activate
   python3 bot.py
   
   Salve: Ctrl+X, Y, Enter

3. Dê permissão:
   chmod +x start.sh

4. Inicie com PM2:
   pm2 start start.sh --name jotinha-bot

5. Configure para iniciar automaticamente:
   pm2 startup
   pm2 save

6. Comandos úteis do PM2:
   pm2 status              # Ver status
   pm2 logs jotinha-bot    # Ver logs
   pm2 restart jotinha-bot # Reiniciar
   pm2 stop jotinha-bot    # Parar
